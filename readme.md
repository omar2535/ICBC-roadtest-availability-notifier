## Setup

**Linux:**

```sh
python3 -m venv .env
source .env/bin/activate

pip install -r requirements.txt
```

**Windows:**

```sh
python3 -m venv .env
.env\Scripts\activate.bat

pip install -r requirements.txt
```

## Running the program

```sh
python main.py
```
